# CSC Restaurant Order Management System

A fully functional restaurant order management web app 
built with pure HTML, CSS, and JavaScript.

## Features
- Multi-table order tracking (Table 1 to 5)
- Real-time order timestamps for every item
- Live dashboard showing all tables simultaneously
- 25+ menu items across Morning, Afternoon, and Dinner
- Automatic bill generation with grand total
- Print bill directly from the browser
- Settle and clear tables after payment
- Fully responsive design

## Technologies Used
- HTML5
- CSS3 (Grid, Flexbox, Media Queries)
- JavaScript (Vanilla, no frameworks)

## How to Run
1. Download or clone this repository
2. Open index.html in any browser
3. No installation or setup required
